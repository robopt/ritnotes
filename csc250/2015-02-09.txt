* Data Alignment
	* Data is said to be aligned if addr % len = 0
	* if read from memory -> must read on aligned
* Address Mode
	* is a math formula using available data to generate an address in memory
		* address in memory -> E.A (Effective address)
	* X - Absolute AM
	* register indirect
		* register contains E.A.
	* displacement
	* indexed
