* DNS Record Types
	* A Record
		* Hostname (Hostname -> IP)
	* CNAME
		* Canonical Name (Alias)
		* Multiple names.
			* EX: www
	* MX
		* Mail Exchange Record
	* NS
		* Name Server
	* PTR
		* Pointer record (used mostly for reverse DNS)
	* RP
		* Responsible Person
	* SOA
		* Start of Authority
	* SRV
		* Service Locator
